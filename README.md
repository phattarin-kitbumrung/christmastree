# christmastree
This repository created just to draw christmastree in multiple language

        *       space = 3 +    [*] + space = 3
       ***      space = 2 +   [***] + space = 2
      *****     space = 1 +  [*****] + space = 1
     *******    space = 0 + [*******] + space = 0
        |       space = 3 +    [|] + space = 3
